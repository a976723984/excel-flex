# Excel-Flex: AI 驱动的智能表格操作应用

Excel-Flex 是一个现代化的 Web 表格应用，其核心亮点是集成了一个强大的 **AI 指令执行引擎**。用户可以通过自然语言下达指令，对表格数据进行复杂的查询、转换、分析和生成操作。本应用旨在探索和展示 AI 在数据处理领域的巨大潜力。

## ✨ AI 核心功能：自然语言指令驱动

应用的灵魂在于顶部的指令输入框和“AI执行”按钮。用户可以在此用日常语言描述其数据处理需求，AI 后端将自动解析、规划并执行这些指令。

### 工作原理

当用户点击“AI执行”后，系统会触发一个复杂而智能的后台工作流：

1.  **指令解析与意图识别**: AI 首先分析用户输入的自然语言指令（Prompt），并结合当前表格的上下文（如工作簿结构、列名等），识别用户的核心意图。

2.  **动态生成执行方案 (Plan)**: 基于意图识别的结果，AI 会动态生成一个结构化的 JSON 执行方案。这类似于 AI 为自己制定了一份详细的操作步骤清单。方案会明确指出需要执行哪种核心操作（`action`），例如：
    *   `filter`: 语义筛选
    *   `merge`: 智能合并（支持精确和模糊匹配）
    *   `analysis`: 数据分析与洞察
    *   `classification`: 内容分类
    *   `generate_column`: 基于上下文生成新列

3.  **分步执行与反馈**: 系统根据生成的方案，调用相应的服务来执行操作。整个执行过程（包括 AI 的思考、决策和操作步骤）会实时显示在弹出的“AI执行流程框”中，为用户提供了极佳的透明度。

### 功能示例

您可以在指令框中尝试以下类型的自然语言指令：

-   **数据生成与扩充**
    -   `“根据A列的公司名称和B列的产品名称，为我生成一段50字左右的产品描述，并放入新列‘产品描述’中”`
    -   `“在D列生成10个100到1000之间的随机整数”`

-   **智能筛选与查询**
    -   `“帮我筛选出‘销售额’列中所有低于平均值的数据”`
    -   `“找出‘用户评论’列中所有提到‘价格’或‘便宜’的评论”`

-   **数据合并与关联**
    -   `“将‘订单表’和‘客户表’通过‘客户ID’进行精确匹配合并”`
    -   `“我有一个‘公司标准名’表和一个‘发票抬头’表，请帮我进行模糊匹配，将它们合并在一起”`

-   **分析与洞察**
    -   `“分析‘销售数据’表中各产品的销售趋势，并给出你的结论”`
    -   `“对‘用户满意度’列进行情感分析，统计正面和负面评价的数量”`

## 🛠️ 技术栈

-   **前端**: HTML5, CSS3, JavaScript (ES6+)
-   **核心AI逻辑**: `src/js/services/aiClient.js` 负责与 AI 模型 API 交互，并调度执行各种复杂任务。
-   **Excel 文件处理**: [SheetJS/xlsx](https://github.com/SheetJS/sheetjs)
-   **开发服务器**: [serve](https://github.com/vercel/serve)

## 🚀 如何开始

1.  **克隆项目**
    ```bash
    git clone <repository-url>
    cd excel-flex
    ```

2.  **安装依赖**
    ```bash
    npm install
    ```

3.  **配置 AI (重要!)**
    -   启动应用后，点击界面右上角的“配置”按钮。
    -   在配置面板中，填入您的 AI 服务提供商的 **API 地址 (API URL)** 和 **API 密钥 (API Key)**。
    -   *本应用兼容 OpenAI, DeepSeek, Anthropic 等多种模型服务。*

4.  **启动开发服务器**
    ```bash
    npm start
    ```
    在浏览器中打开 `http://localhost:3000`，即可开始通过自然语言与您的数据进行交互。
