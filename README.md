# Excel-Flex

一个功能强大、性能卓越的网页版 Excel 表格应用，旨在提供流畅、直观的电子表格操作体验。借助虚拟化渲染技术，Excel-Flex 能够轻松处理和展示海量数据，同时保持界面的高响应度。

## ✨ 核心功能

- **Excel 文件导入**: 支持通过对话框选择并导入 `.xlsx` 格式的 Excel 文件，快速将数据加载到网页中。
- **虚拟化渲染**: 采用先进的虚拟滚动技术，仅渲染当前可见区域的行和列，确保在处理数十万行数据时依然保持流畅的滚动和操作体验。
- **动态状态栏**: 界面底部提供一个信息丰富的状态栏，实时显示以下内容：
  - **总行数**: 清晰了解当前工作表的总数据量。
  - **可见范围**: 动态展示当前屏幕上可见的行和列的范围。
  - **单元格选择**: 显示当前选中单元格的坐标或范围。
- **可交互的表格**: 
  - 支持单元格、整行、整列的选择。
  - 支持双击编辑单元格内容。
- **多工作簿和工作表管理**: 
  - 支持在多个工作簿（Workbook）之间切换。
  - 支持在同一工作簿内创建、切换和删除多个工作表（Sheet）。

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **Excel 处理**: [SheetJS/xlsx](https://github.com/SheetJS/sheetjs) 用于解析和处理 `.xlsx` 文件。
- **开发服务器**: [serve](https://github.com/vercel/serve) 用于快速启动本地开发环境。

```

```

## 🚀 如何开始

1.  **克隆项目**
    ```bash
    git clone <repository-url>
    cd excel-flex
    ```

2.  **安装依赖**
    ```bash
    npm install
    ```

3.  **启动开发服务器**
    ```bash
    npm start
    ```
    该命令会启动一个本地服务器（通常在 `http://localhost:3000`），在浏览器中打开该地址即可看到应用界面。

4.  **使用**
    - 点击“导入 Excel”按钮，选择一个 `.xlsx` 文件进行加载。
    - 使用鼠标滚轮或滚动条来浏览数据，体验虚拟滚动的效果。
    - 观察底部状态栏的变化。
